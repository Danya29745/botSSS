# ⚡️ HalvexVPN Bot

Telegram-бот для продажи VPN-подписок. Построен на `aiogram 3` + `SQLAlchemy (async)` + `SQLite/PostgreSQL`.

---

## 🚀 Быстрый старт

### 1. Клонируй репозиторий
```bash
git clone https://github.com/твой-ник/halvex-vpn-bot.git
cd halvex-vpn-bot
```

### 2. Установи зависимости
```bash
pip install -r requirements.txt
```

### 3. Настрой переменные окружения
```bash
cp .env.example .env
nano .env   # заполни BOT_TOKEN и ADMIN_IDS
```

### 4. Запусти бота
```bash
python bot.py
```

---

## 🌐 Деплой на хостинг

### Переменные окружения (указать в панели хостинга):

| Переменная | Описание |
|---|---|
| `BOT_TOKEN` | Токен бота от @BotFather |
| `ADMIN_IDS` | Telegram ID админов через запятую |
| `DATABASE_URL` | URL базы данных |
| `REFERRAL_BONUS_REFERRER` | Бонус рефереру (₽) |
| `REFERRAL_BONUS_NEW_USER` | Бонус новому юзеру (₽) |
| `XUI_HOST` | Адрес 3x-UI панели |
| `XUI_USERNAME` | Логин панели |
| `XUI_PASSWORD` | Пароль панели |

## 👥 Реферальная система

- Каждый пользователь получает уникальный реф-код при регистрации
- Ссылка: `https://t.me/БОТ?start=КОД`
- При регистрации по ссылке:
  - Рефереру начисляется бонус (по умолчанию **50 ₽**)
  - Новому пользователю начисляется бонус (по умолчанию **30 ₽**)
- Начисление — мгновенное, на баланс в боте

## 🛡 Админ-панель (`/admin`)

- 📢 **Рассылка** — отправить сообщение всем пользователям
- 👥 **Все пользователи** — список с балансами
- 🆕 **Новые за 24ч** — кто зарегистрировался сегодня
- 🎫 **Тикеты поддержки** — открытые вопросы, ответ прямо из панели
- 📊 **Статистика** — общие цифры
- 🔎 **Найти юзера** — по ID кабинета или TG ID
- 💸 **Начислить баланс** — вручную
- 🚫 **Бан / Разбан** — блокировка пользователя
